Here's a keyboard guide that includes some commonly used keyboard shortcuts:



  • Function Keys (F1-F12):


    • F1: Help or context-sensitive help.
    F2: Rename a selected file or folder.
    F5: Refresh the current page or window.
    F11: Toggle full-screen mode in web browsers.

  • Modifier Keys:


    • Ctrl (Control key): Ctrl+C: Copy selected text or files.
    Ctrl+X: Cut selected text or files.
    Ctrl+V: Paste copied or cut text or files.
    Ctrl+Z: Undo the last action.
    Ctrl+A: Select all items in a document or folder.

  • Shift key:


    • Shift+Delete: Delete selected item permanently without moving it to the Recycle Bin or Trash.
    Shift+Insert: Paste copied or cut text or files.

  • Alt key:


    • Alt+Tab: Switch between open applications or windows.
    Alt+F4: Close the active window or application.
    Alt+Enter: Open properties of the selected item.

  • Windows key (Windows OS) or Command key (Mac OS):


    • Win/Command key: Open or close the Start menu or Launchpad.
    Win/Command+Tab: Switch between open applications or windows.
    Win/Command+D: Create a new desktop (Windows OS) or enter Mission Control (Mac OS).
    Win/Command+L: Lock the computer or screen.
    Win/Command+E: Open File Explorer (Windows OS) or Finder (Mac OS).

  • Text Formatting Shortcuts:


    • Ctrl/Command+B: Bold selected text.
    Ctrl/Command+I: Italicize selected text.
    Ctrl/Command+U: Underline selected text.
    Ctrl/Command+Shift+V: Paste and match the formatting of the destination text.

  • Web Browsing Shortcuts (in web browsers like Chrome, Firefox, or Edge):


    • Ctrl/Command+T: Open a new tab.
    Ctrl/Command+W: Close the current tab.
    Ctrl/Command+Tab: Switch between open tabs.
    Ctrl/Command+L: Move the cursor to the address bar.
    Ctrl/Command+R: Refresh the current page.
    Ctrl/Command+F: Open the find/search bar.

  • Miscellaneous Shortcuts:


    • Ctrl/Command+S: Save the current document or file.
    Ctrl/Command+P: Print the current document or file.
    Ctrl/Command+Z: Undo the last action.
    Ctrl/Command+Shift+Esc: Open the Task Manager (Windows OS) or Activity Monitor (Mac OS).
    Ctrl/Command+Shift+N: Create a new folder.
    Ctrl/Command+Shift+T: Reopen the last closed tab (in web browsers).
    Ctrl/Command+Shift+Delete: Open the Clear browsing data/Clear history window (in web browsers).
